# devroast

> paste your code. get roasted.

Plataforma para submeter código e receber críticas — honestas ou no modo "roast" máximo. Feito durante o **NLW da RocketSeat**.

## Funcionalidades

- **Enviar código** — Cole seu código e receba uma análise
- **Modo Roast** — Ative para críticas sarcásticas e sem filtro
- **Shame Leaderboard** — Veja os códigos mais roasted da comunidade

## Como rodar

```bash
# Instalar dependências
npm install

# Rodar em desenvolvimento
npm run dev

# Build para produção
npm run build
```

Acesse: http://localhost:3000

## Tech Stack

Next.js • React • Tailwind CSS • TypeScript
